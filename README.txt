TD7 Pokemons Héritage, polymorphisme, collections.

Lancer la méthode Main de la classe Main pour effectuer un combat entre les 2 joueurs.

Pour accéder à notre extension du TP lancer la méthode Main de pokedexGUI

Notre repository : https://github.com/mickaelicoptere/TD7_Pokemon

Librairies utilisées : 
- JUnit 4 pour les test unitaires.
- SI_VOX pour la synthèse vocale du pokédex (auteur : http://users.polytech.unice.fr/~helen/ )
- jansi-1.16 pour la couleur dans le terminal.
- JFoenix (lien du repo : https://github.com/jfoenixadmin/JFoenix )
